# 📊 Data Jobs Analytics Dashboard

An interactive Power BI dashboard built to analyze the global data job market, helping users explore salary trends, in-demand skills, job availability, and employment patterns across different data-related roles.

---

## 📌 Project Overview

The **Data Jobs Analytics Dashboard** provides a comprehensive analysis of data-related job postings. Using Power BI, Power Query, DAX, and SQL concepts, the dashboard transforms raw job posting data into interactive visualizations that help identify industry trends and support data-driven career decisions.

The dashboard enables users to filter information by job title, country, work schedule, and degree requirements while exploring salary distributions and the most in-demand technical skills.

---

## 🎯 Objectives

- Analyze global data job postings.
- Identify the highest-paying data careers.
- Discover the most in-demand technical skills.
- Compare yearly and hourly salaries across job roles.
- Provide an interactive dashboard for career exploration.

## 📈 Dashboard Features

### 📌 KPI Cards

The dashboard displays key business metrics including:

- Total Job Postings
- Median Yearly Salary
- Median Hourly Salary
- Average Skills Required per Job

---

### 📊 Interactive Visualizations

The report includes:

- Top Skills by Job Count
- Median Salary by Job Title
- Interactive Filters (Slicers)
- Dynamic KPI Cards
- Comparative Salary Analysis

Users can interact with slicers to instantly update all visuals.

---

## 🔍 Key Insights

- Data Analyst and Data Engineer roles have the highest number of job postings.
- Salary varies significantly across different job titles.
- SQL, Python, and Power BI are among the most frequently requested skills.
- Interactive filters allow detailed analysis by job role and other criteria.
